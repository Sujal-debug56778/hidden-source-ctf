
Hidden Source Flag - CTF Challenge

🔍 Challenge Overview

This is a beginner-friendly Capture The Flag (CTF) challenge based on web security fundamentals. The goal is to find a hidden flag embedded within the webpage source code.

⚙️ Setup Instructions

1. Download or clone the repository.
2. Open the file "index.html" in any web browser.
3. View the page content.

📖 Challenge Story / Scenario

You are given access to a simple webpage created by a careless developer. The page looks normal, but the developer might have accidentally left some sensitive information behind.

🚩 Flag Format

GCS{...}

📍 Flag Location / Logic

The flag is hidden inside an HTML comment in the source code of the webpage.

🧠 Expected Solution Path

1. Open the webpage in a browser.
2. Right-click and select "View Page Source" or press "Ctrl + U".
3. Look for hidden comments in the HTML.
4. Identify the flag inside the comment.

🎯 Learning Objectives

- Learn how to inspect webpage source code
- Understand common beginner-level web vulnerabilities
- Recognize risks of exposing sensitive data in HTML comments
- Use browser developer tools effectively

✅ Learning Outcomes

After completing this challenge, participants will be able to:

- Analyze web page structures
- Identify hidden data in frontend code
- Apply basic web security awareness
